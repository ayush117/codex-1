# BloomDesk landing app

A lightweight static app inspired by modern SaaS landing pages (including usesprout.com style patterns):

- Hero with value proposition and waitlist signup.
- Feature grid and “how it works” sections.
- Pricing card and FAQ.
- Responsive layout for mobile and desktop.

## Run locally

```bash
python3 -m http.server 4173
```

Then open http://localhost:4173.
